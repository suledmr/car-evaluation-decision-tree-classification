# Car Evaluation Classification using Decision Tree

This project builds a machine learning model to classify cars based on evaluation criteria using the Decision Tree algorithm.

## Project Overview

The goal is to predict car evaluation categories based on features such as buying price, maintenance cost, number of doors, capacity, luggage boot size, and safety.

## Steps

- Data loading
- Data preprocessing
- Encoding categorical variables
- Exploratory Data Analysis (EDA)
- Model training (Decision Tree)
- Model evaluation

## Dataset

Dataset source: Kaggle - Car Evaluation Dataset

Dataset link: <https://www.kaggle.com/datasets/elikplim/car-evaluation-data-set>

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
